üöÄ Getting Started

The Release Generator 2.0 is a comprehensive tool for creating Hearts of Iron IV event code to establish puppet states. This tool generates properly formatted event files, localization, and decision categories.

Quick Start Workflow:

  1. Set your Master Country Tag and Puppet Ideology
  2. Add countries with their target tags and state IDs
  3. Configure government and leader settings (optional)
  4. Adjust political party popularity
  5. Add focus requirements (optional)
  6. Generate your event code

‚öôÔ∏è Basic Setup

üéØ Master Country Tag

The country tag of the nation that will be establishing the puppet states (e.g., "GER", "JAP", "ITA").

Example: GER for Germany

üëë Puppet Ideology

The political ideology that puppet states will adopt. This affects their ruling party and political orientation.

fascism, communism, democratic, neutrality

Impact: Determines the puppet's political system and available laws

üí° Tip: The puppet ideology dropdown contains all valid KX ideologies.

üåç Country Management

Each country you want to establish as a puppet needs specific configuration.

üÜï Adding Countries

Use the "Add Country" button to create new country entries. Each country has three main sections:

üèõÔ∏è Country Info

  • Target Tag: The country code for the puppet state (e.g., "POL", "CZE")
  • Nation Full Name: Display name for the country
  • State Core IDs: States where the puppet will have cores
  • Required State IDs: States needed for event activation

‚öñÔ∏è State ID Types

State Core IDs: Territories the puppet will claim as their own

Required State IDs: States your master nation or puppet must control to trigger the event

# Example: State Core IDs: 1,2,3,4 # Puppet gets cores here Required State IDs: 1,2 # Master must own these

üîß Advanced Options

"Use OR block" Checkbox

When checked, ONLY ONE of the required states is needed to activate the event.

Unchecked (Default):

Checked:

available = { OR = { 1 = { owner = { OR = { original_tag = GER is_subject_of = GER } } } 2 = { owner = { OR = { original_tag = GER is_subject_of = GER } } } } OR = { 1 = { NOT = { is_core_of = GER } } 2 = { NOT = { is_core_of = GER } } } }
